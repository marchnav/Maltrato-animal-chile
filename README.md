# Alto al maltrato animal en Chile

Sitio estático de ejemplo creado para visibilizar la realidad del maltrato animal en Chile,
promover la empatía y entregar ideas concretas de acción responsable.

> **Importante:** El contenido es informativo y general. No reemplaza asesoría legal, veterinaria
> ni profesional. La legislación y los canales de denuncia pueden cambiar con el tiempo.

## Estructura del proyecto

```text
maltrato_animal_chile_site/
├─ index.html
├─ css/
│  └─ styles.css
├─ js/
│  └─ main.js
└─ assets/
   └─ (imágenes opcionales, ver más abajo)
```

- `index.html`: página principal, ya estructurada y con secciones:
  - Hero / portada
  - Realidad del maltrato animal
  - Tipos de maltrato y señales de alerta
  - ¿Qué puedo hacer?
  - Recursos y organizaciones
- `css/styles.css`: estilos base, diseño responsive y enfoque de contraste alto.
- `js/main.js`: pequeñas mejoras de experiencia (menú móvil y scroll suave).
- `assets/`: carpeta pensada para imágenes e ilustraciones.

## Cómo usar este proyecto

### 1. Verlo en local (solo con navegador)

1. Descarga o copia todos los archivos manteniendo la misma estructura de carpetas.
2. Abre `index.html` con tu navegador (doble clic o “Abrir con…”).
3. Deberías ver el sitio tal como se diseñó.

### 2. Publicarlo en GitHub Pages (modo simple, incluso desde el celular)

1. Crea un repositorio nuevo en GitHub (por ejemplo: `maltrato-animal-chile`).
2. Sube todos los archivos y carpetas manteniendo la estructura anterior.
3. En el repositorio, ve a **Settings → Pages**.
4. En “Branch”, elige `main` (o `master`) y la carpeta raíz `/ (root)`.
5. Guarda los cambios. GitHub generará una URL pública en pocos minutos.

### 3. Imágenes sugeridas

Dentro de `assets/` puedes agregar imágenes creadas por ti o generadas con IA,
respetando siempre derechos de autor y evitando contenido morboso o sensacionalista.

Algunas ideas de prompts (para usar en herramientas de generación de imágenes):
- “Ilustración suave de un perro rescatado, con mirada tranquila, estilo semi-realista, colores cálidos, sin violencia ni sangre.”
- “Gato en un hogar responsable, con juguetes y cama cómoda, luz natural, estilo fotográfico.”
- “Ilustración educativa de una comunidad apoyando la tenencia responsable de animales de compañía, ambiente urbano latinoamericano.”

Luego, referencia esos archivos desde `index.html` con etiquetas `<img>` colocando textos alternativos (`alt`) descriptivos.

## Adaptación a tu realidad local

La sección de “Recursos y organizaciones” está escrita de forma genérica para que puedas:
- Reemplazar el texto con nombres de organizaciones reales de tu ciudad o región.
- Agregar enlaces a sitios oficiales de gobierno y colegios veterinarios.
- Incorporar material educativo propio o de campañas locales.

Siempre revisa la legislación vigente y las recomendaciones de fuentes oficiales.
